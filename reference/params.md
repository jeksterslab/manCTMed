# Simulation Parameters

Simulation Parameters

## Usage

``` r
data(params)
```

## Format

A dataframe with 30 rows and 3 columns:

- taskid:

  Simulation Task ID.

- n:

  Sample size.

- dynamics:

  Dynamics. `0` for original drift matrix, `-1` for near-neutral
  dynamics, and `1` for stronger damping.
