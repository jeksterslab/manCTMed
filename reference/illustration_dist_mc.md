# Illustration Sampling Distribution (Monte Carlo Method)

Illustration Sampling Distribution (Monte Carlo Method)

## Usage

``` r
data(illustration_dist_mc)
```

## Format

A matrix with 20000 rows and 15 columns:

- phi_xx:

  Elements of the drift matrix.

- sigma_xx:

  Elements of the process noise covariance matrix.
