# Illustration Sampling Distribution Standardized Total, Direct, and Indirect Effects - Time Interval of 1

Illustration Sampling Distribution Standardized Total, Direct, and
Indirect Effects - Time Interval of 1

## Usage

``` r
data(illustration_dist_med_std)
```

## Format

A matrix with 1000 rows and 27 columns:

- total:

  Total effect.

- direct:

  Direct effect.

- indirect:

  Indirect effect.
